Funções das Tags(Voltar)

As tags possuem várias funções no HTML, uma delas é organizar o conteúdo do HTML, as Tags também criam conteúdos e elementos nas página da web e identificam e limita counteúdos das páginas.

Exemplo de Tags(Voltar)

Head:Head é o counteúdo do Html que será interpretado pelo navegador, o usuário da web não vera essas informações.

Body:Body é o counteúdo que será enviado para paginá na web, links, parágrafos, o usuário pode ver e interagir com elas.

Style:Style é a tag usada para fazer estilos de CSS para o HTML.
